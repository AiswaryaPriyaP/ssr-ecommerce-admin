# SSR E-Commerce Product Management Dashboard

## Project Overview
This project is a Server-Side Rendered (SSR) e-commerce admin dashboard built using Next.js App Router.  
It enables administrators to manage products, upload images, and view basic analytics while demonstrating SSR concepts such as fresh data on every request and improved performance.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Features
- Server-side rendered product listing
- Product management (Add and View products)
- Image upload using file input (Cloudinary)
- Form validation using Zod and React Hook Form
- Dummy admin authentication
- Sales & analytics chart (products by category)
- Clean admin dashboard UI with sidebar navigation

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Tech Stack
Frontend: Next.js (App Router), React, TypeScript, Tailwind CSS  
Backend: Next.js API Routes, MongoDB Atlas, Mongoose  
Validation: React Hook Form, Zod  
Image Storage: Cloudinary  
Charts: Chart.js / Recharts
